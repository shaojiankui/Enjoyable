What's New in 1.2

  • Automatic switching for applications with ambiguous bundle names and executable filenames (e.g. "wine.bin") has improved. You can now name a mapping after an application's primary window title.
  • Compatibility with OS X 10.9's power-saving features (App Nap, etc.) has been improved. Now works at all on OS X 10.10.
  • Upgrades no longer occur automatically. This is due to privacy concerns and Apple repeatedly breaking their code signing infrastructure.
